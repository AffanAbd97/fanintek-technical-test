## Note

-   **Database Version : PostgreSQL 15.4**
-   **Node Version : 20.9.0**


# Local Installation For Test Script

-   run `npm install `
-   copy .env.example to .env
-   run `cp .env.example .env`
-   configure the database connection on .env
-   run `npx prisma migrate dev`
-   run `npm run start`
